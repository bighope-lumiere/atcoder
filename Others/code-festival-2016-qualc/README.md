## B - K個のケーキ / K Cakes

一番個数が多いケーキを並べて、それらの間にさえしっかり入れられればOK。
一番個数が多いケーキの間は max(a) - 1 個あり、個々に入れられる要素の個数は sum(a) - max(a) 個である。
しっかり入れられる場合は 0 を出力すればよいので、結局答えは max(0, max(a) - 1 - (sum(a) - max(a))) になる。
